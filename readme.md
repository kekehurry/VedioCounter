# Vedio Counter V1.0


### Install

* download yolov3 from "https://github.com/OlafenwaMoses/ImageAI/releases/download/3.0.0-pretrained/yolov3.pt/"
* put the "yolov3.pt" in the same folder as the main.py
* pip install -r requirements.txt


### Usage

`python main.py`
